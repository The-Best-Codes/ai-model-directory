# Docs frontmatter

Docs pages are MDX files in `packages/web/src/docs`. Routing is file-based:

- `index.mdx` becomes `/docs`
- `data-layout.mdx` becomes `/docs/data-layout`
- `guides/example.mdx` becomes `/docs/guides/example`

Each page can define simple frontmatter.

```mdx
---
title: Page title
description: Short page description for metadata
order: 10
---

# Page title
```

The sidebar is generated from the files automatically. Use `order` to place a page earlier or later in the sidebar.
