# AI Model Directory docs

AI Model Directory is the most comprehensive automatically updated catalog of AI providers and models. It tracks provider metadata, model IDs, pricing, context and output limits, modalities, feature flags, release dates, and more.

## What you can do

- [Browse models](/list) on the website.
- Use `data/all.json` or `data/all.min.json` directly.
- Install the `ai-model-directory` npm package for typed helpers.
- Contribute provider adapters and metadata fixes.
