# AI Model Directory docs

AI Model Directory is a large, open data set of AI providers and their models. It tracks IDs, names, pricing, limits, features, and provider metadata in one place.

## What you can do

- [Browse models](/list) on the website.
- Install the npm package and use the generated JSON-like data in your app.
- Add or update providers by editing the provider files in the repository.
